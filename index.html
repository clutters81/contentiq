About ContentIQ

Built on the Hybrid Content Ecosystem — the C³ Method (Content → Conversations → Consultations). Content is structured across 3 buyer stages: Awareness, Information, Action.

To activate AI generation: set ANTHROPIC_API_KEY as a Vercel environment variable.
